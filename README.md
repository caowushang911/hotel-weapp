# 酒店预订系统-微信小程序
该仓库为酒店预订系统的微信小程序，其Vue后台和Node.js服务端如下：
> [Vue前台](https://github.com/xkcease/hotel-management)  

> [Node.js服务端](https://github.com/xkcease/hotel-server)

## 功能
1.  登录 
2.	首页
3.	预订页
4.	详细 
5.	我的
6.	订单列表
7.  订单详细
8.  确认预订
